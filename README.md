# 🌍 StockSense Africa

**AI-powered Inventory & Procurement Intelligence for Small Retail Businesses in Africa**

---

## 📌 Overview

StockSense Africa is a smart inventory management and procurement decision-support system designed for small and medium retail businesses.

It helps shop owners:
- Track stock levels in real-time  
- Identify products that need urgent reordering  
- Compare supplier prices  
- Understand revenue performance  
- Generate AI-powered procurement recommendations  

---

## 🚀 Key Features

### 📊 Dashboard (Overview)
- Revenue trends over time  
- Total units sold  
- Product performance breakdown  
- High-level business KPIs  

### 📦 Inventory Intelligence
- Stock health monitoring (Critical / Warning / Safe)
- Days-of-stock estimation
- Automatic reorder alerts
- Product-level sales analysis  

### 🏪 Procurement Tracker
- Supplier price comparison system  
- Best price identification  
- Add and manage suppliers dynamically  

### 🤖 AI Insights Engine
- Autonomous procurement agent powered by LLMs  
- Generates actionable reorder decisions  
- Produces supplier-ready messages  
- Business health scoring  

---

## 🧠 AI Agent Logic

The system uses an AI agent that:
1. Reads inventory data
2. Classifies stock risk levels
3. Evaluates supplier options
4. Generates procurement actions
5. Outputs structured business recommendations

## 🎯 Problem It Solves

Small retail businesses often:
- Overstock slow-moving items  
- Run out of fast-moving stock  
- Lack data-driven procurement decisions  
- Have no visibility into supplier pricing efficiency  

StockSense Africa solves this by turning raw sales data into **actionable procurement intelligence**.

## 📈 Impact

- Reduces stockouts  
- Improves cash flow efficiency  
- Optimizes supplier selection
